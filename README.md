# Elinknest — Education Marketplace Platform

Education marketplace platform connecting students, tutors, schools and universities.

The project included a web platform, mobile student app, admin panel, tutor profiles, WhatsApp verification, payments, role-based access and backend automation.

## Technical challenges

### Role-based platform logic

The project had several user types: students, tutors, schools, universities and admins.

I implemented role-based access logic and separated user flows depending on the user type.

### Tutor activation automation

Tutors and organizations had paid visibility periods.

I implemented backend logic to check payment expiration dates and update profile activity status automatically.

### WhatsApp Business API integration

The platform required WhatsApp verification and messaging logic.

I integrated WhatsApp Business API, handled verification flow, API communication and production setup.

### Admin panel and storage access

The admin panel needed access to user data and uploaded images.

I implemented admin logic and storage access rules so admins could manage platform content.
